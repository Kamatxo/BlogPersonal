¿ Sistemas Operativos (SO) ?

UTM como todas las Máquinas Virtuales necesitan de una imagen de disco de un sistema opertaivo donde poder arrancar e instalarlo.

Los archivos de imágenes pueden ser de diferentes tipos: *.iso, *.qcow2, *.vdi, etc.. dejo aquí el enlace con la información, aunque aparezca qemu, es lo mismo.

¿ De donde podemos obtener los discos para instalar los sistemas operativos ?

Qemu nos brinda la posibilidad de poder descargarnos por ejemplo, el Ubuntu (que es la página de descarga de Ubuntu ARM Server). Tambien descargarnos archivos de UTM, que al final no son mas que los ajustes de una máquina virtual (por ejemplo esta para el Windows XP entre otros)

Los sistemas Operativos Ubuntu y Windows ARM podemos descargarlos de las páginas oficiales

¿ Porque descargar la versión ARM64 y no el x64 ?

La versión ARM, aunque es más complicada de instalar, al coincidir la arquitectura con la de tu equipo (el chip de Apple M1 es aqruitectura ARM64) es mas fluido (en AMD es muy lento) y es mucho más eficiente que la versión x64.

En todo caso si quieres descargarte la versión oficial de Ubuntu x64, te dejo aquí la guía pues tendrás que hacer una comprobación por si el archivo no esta corrupto.
